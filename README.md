# Miniflux Reader

一个基于 React Router 7 构建的 Miniflux Web 客户端。

## 开发

```bash
pnpm install
pnpm dev
```

默认开发地址为 `http://localhost:5173`。

## 常用命令

```bash
pnpm dev
pnpm build
pnpm test
pnpm typecheck
```

## 说明

- 路由定义位于 `app/routes.ts`
- 页面模块位于 `app/pages`
- 共享组件位于 `app/components`
- Miniflux API 相关代码位于 `app/lib/miniflux`
